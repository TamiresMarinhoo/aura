# Studio Aura — Landing Page

Landing page de alto padrão para um escritório fictício de arquitetura e design de interiores, desenvolvida durante uma imersão de dois dias com a **DevQuest** sobre como projetar sites utilizando inteligência artificial.

## Sobre o projeto

O conceito criativo é o **"Luxo Discreto"** — uma experiência visual inspirada em galerias de arte e hotéis boutique. A página apresenta o portfólio da arquiteta Marina Vasconcelos com foco em sofisticação, minimalismo e atenção extrema aos detalhes.

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla

## Funcionalidades

- Hero section com vídeo de fundo
- Seção de metodologia com suporte visual em vídeo
- Galeria de portfólio com efeito parallax
- Seção sobre a fundadora
- Depoimentos de clientes
- Formulário de contato com validação
- Menu mobile com acessibilidade (ARIA)
- Header com comportamento sticky ao scroll
- Navegação ativa por seção (section spy)
- Layout inteiramente responsivo (mobile-first)

## Design System

A identidade visual completa está documentada em [`DESIGN.md`](./DESIGN.md), incluindo paleta de cores, tipografia (EB Garamond + Manrope), grid e componentes.
